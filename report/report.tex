\noindent\textbf{摘\quad 要：}
本实验围绕“边缘草图到商品图像”的图像到图像翻译任务，基于 Pix2Pix 条件生成对抗网络实现了鞋类草图到彩色商品图像的生成系统。模型采用 U-Net 作为生成器、PatchGAN 作为判别器，并联合使用对抗损失和 L1 重建损失进行训练。实验使用 edges2shoes 数据集，在 CPU 环境下抽取 100 对训练图像训练 5 个 epoch，并在 20 对验证图像上进行推理和评价。实验结果显示，生成器总损失由第 1 个 epoch 的 38.2775 下降到第 5 个 epoch 的 19.9973；在验证图像上得到 Mean L1 为 0.3447、PSNR 为 7.08、SSIM 为 0.3414。可视化结果表明，模型能够保持鞋子的大致轮廓和位置关系，但颜色稳定性与局部纹理仍受小样本训练规模限制。除模型训练外，本项目还整理了推理、评价、可视化、Gradio 演示界面、LaTeX 报告和 GitHub 开源仓库，形成了一个可复现实验流程。

\section{引言}
生成对抗网络（Generative Adversarial Network, GAN）是深度生成模型中的重要方法。GAN 由生成器和判别器组成：生成器尝试合成接近真实分布的样本，判别器则判断输入样本来自真实数据还是生成器。二者在训练中形成对抗关系，使生成器逐步学习数据分布。相比传统的判别式模型，GAN 更强调从数据中学习“如何生成”样本，因此在图像生成、图像修复和图像翻译等任务中具有较强的代表性。

本实验关注的是有条件的图像生成任务。对于鞋类草图到商品图像的转换，输入边缘图已经提供了鞋子的外部轮廓和主要结构，模型需要在该条件下补充颜色、明暗和局部纹理。Pix2Pix 是一种面向成对图像翻译任务的条件 GAN 方法，适合处理边缘图到照片、标签图到街景等监督式图像转换问题。因此，本实验选择 Pix2Pix 作为主要模型，在 edges2shoes 数据集上完成“边缘草图 $\rightarrow$ 鞋类商品图像”的小样本实验，并将训练、推理、评价和展示过程整理为完整项目。

\section{相关工作}
Goodfellow 等人提出的 GAN\cite{goodfellow2014gan} 通过最小最大博弈建立生成模型，为后续图像生成研究提供了基础框架。普通 GAN 通常从随机噪声出发生成样本，而条件 GAN 在生成过程里引入额外条件，使输出受到类别、文本或图像等信息控制。对于图像翻译任务，条件输入通常就是待转换的图像。

Pix2Pix\cite{isola2017pix2pix} 将条件 GAN 用于成对图像到图像翻译，并引入 U-Net 生成器和 PatchGAN 判别器。该方法的一个特点是同时使用对抗损失和 L1 重建损失：前者鼓励生成图像更真实，后者约束输出与目标图像在像素层面接近。CycleGAN\cite{zhu2017cyclegan} 则主要用于非成对图像数据，通过循环一致性约束学习两个域之间的映射。由于 edges2shoes 数据集提供了边缘图和鞋子照片的成对样本，本实验采用 Pix2Pix 比采用非成对方法更直接，也便于分析训练目标与实验结果。

\section{方法}
\subsection{任务定义}
设输入边缘草图为 $x$，目标商品图像为 $y$。实验目标是训练生成器 $G$，使输出 $G(x)$ 尽可能接近真实目标图像 $y$。判别器 $D$ 的输入是条件图像与目标图像的拼接，即 $(x,y)$ 或 $(x,G(x))$，输出为局部 patch 的真伪判断。

\subsection{生成器结构}
生成器采用 U-Net 编码器--解码器结构。编码器通过逐层下采样提取高层特征，解码器通过逐层上采样恢复图像空间尺寸。U-Net 的跳跃连接将浅层边缘信息传递到解码阶段，有助于保留鞋底、鞋面边界和整体外形。对于草图到图像的转换任务，这种结构比普通编码器--解码器更适合保持输入轮廓。

\subsection{判别器结构}
判别器采用 PatchGAN 结构。PatchGAN 并不直接对整张图像给出一个真伪标量，而是对图像局部区域进行判断。这样可以让判别器更关注局部纹理、边缘连续性和颜色块的自然程度，同时保持较少的参数量。训练时，真实图像对被标记为正样本，生成图像对被标记为负样本。

\subsection{损失函数}
生成器损失由对抗损失和 L1 重建损失组成：
\begin{equation}
\mathcal{L}_{G}=\mathcal{L}_{GAN}(G,D)+\lambda_{L1}\mathcal{L}_{1}(G),
\end{equation}
其中 $\lambda_{L1}=100$。较大的 L1 权重可以让训练前期更加稳定，并鼓励输出图像与成对标注图像保持一致。判别器则通过区分真实图像对和生成图像对进行优化。

\section{数据集与实验设置}
实验使用 edges2shoes 数据集。该数据集中每张样本图像由左右两部分拼接而成，一侧为鞋子的边缘图，另一侧为对应的鞋子照片。程序读取图像后从中间切分，并按照 AtoB 方向将边缘图作为输入、真实照片作为目标图像。为保证实验在普通 CPU 环境下也能运行，本次实验从原始训练集中抽取 100 对图像，并使用 20 对验证图像进行推理和评价。

评价指标使用 Mean L1、PSNR 和 SSIM。Mean L1 反映生成图像与目标图像之间的平均像素误差；PSNR 常用于评价重建质量；SSIM 用于衡量图像结构相似性。除定量指标外，实验还保存输入草图、目标图像和生成图像的三列对比图，用于观察模型是否保持轮廓以及是否生成合理的颜色区域。

\section{系统实现}
项目使用 PyTorch\cite{pytorchdocs} 实现。数据集模块负责加载成对图像、切分左右半图、调整尺寸，并将像素值归一化到 $[-1,1]$。模型模块包括 U-Net 生成器、PatchGAN 判别器、损失函数以及 Pix2Pix 训练封装。训练脚本会在每个 epoch 后记录 G loss、D loss、GAN loss 和 L1 loss，并保存样例生成图。

推理阶段只加载训练好的生成器，对验证图像中的边缘草图进行转换，并将输出反归一化到普通图像格式。评价脚本按照文件名匹配 generated 与 target 图像，计算 Mean L1、PSNR 和 SSIM，并同时保存 JSON 与 Markdown 格式的指标结果。项目还提供了 Gradio 演示界面，用于上传草图并展示生成结果。代码、报告和轻量实验结果已整理到 GitHub 仓库，项目地址见标题区。

\section{结果分析}
从图 \ref{fig:inference} 可以观察到，生成图像大体跟随了输入草图的鞋子轮廓，鞋底和鞋面的空间位置基本对应。这说明 U-Net 中的跳跃连接对保留结构信息是有帮助的。另一方面，生成结果的颜色不够稳定，局部纹理较为模糊，部分区域会出现灰色或平滑块。这与训练样本较少、训练轮数较短以及 L1 损失的平均化倾向有关。

判别器损失在 0.43 到 0.59 之间波动，没有出现明显崩溃，但小样本条件下的判别器很难学习到充分多样的局部纹理分布。对抗损失能够改善局部真实感，但当训练数据不足时，它也容易受到样本分布有限的影响。因此，本实验中模型更擅长恢复大致形状，而不是生成清晰且稳定的鞋面材质。

\section{不足与改进方向}
本实验的主要不足是训练规模较小。100 对样本不足以覆盖鞋子外观的多样性，5 个 epoch 也不足以让 GAN 训练充分收敛。此外，Pix2Pix 依赖成对数据，如果应用到没有配对图像的场景，需要考虑 CycleGAN 等非成对图像翻译方法。

后续可以从三个方面改进。第一，在 GPU 环境下使用更多样本和更多训练轮数；第二，在损失函数中加入感知损失或特征匹配损失，以改善纹理细节；第三，与 CycleGAN、CUT 等方法进行对比实验，分析不同图像翻译方法在草图到商品图像任务上的差异。

\section{总结}
本实验实现了基于 Pix2Pix 的鞋类边缘草图到商品图像生成系统。通过训练 U-Net 生成器和 PatchGAN 判别器，模型能够根据输入草图生成大致符合轮廓的鞋子图像。实验结果表明，在 100 对训练样本和 5 个 epoch 的设置下，损失具有下降趋势，但生成图像仍存在颜色不稳定和纹理模糊的问题。通过本次实验，我对条件 GAN、Pix2Pix 模型结构、成对图像数据处理以及生成模型评价指标有了更直观的理解，也完成了一个包含训练、推理、评价、展示和项目发布的完整深度学习流程。
